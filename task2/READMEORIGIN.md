# Go Hello World Application Deployment with Kubernetes

This task focuses on deploying a simple Go "Hello World" http server application to a Kubernetes cluster. The challenge involves identifying and fixing issues along the way.

## Submission (keep it simple and complete)
Provide a single folder that contains:
- Code: All manifests, templates, and fixes you wrote
- Docs: A README with documentation (issues, solutions, improvements, etc.)
- Requests/Response: Include the CLI request commands along with the responses from all 3 endpoints for each solution (raw manifests and Helm chart) in the README, or provide them as screenshots.

## Dependencies
- Go
- Docker Desktop 4.40+ with Kubernetes enabled
- kubectl and Helm

## Evaluation (what we look for)
- Ability to identify and fix issues
- Understanding of Go, containerization, and Kubernetes concepts
- Quality and completeness of the solution and documentation
